>>> Mall_Customer_Segmentation

>> Project Overview
This project performs customer segmentation using Python and K-Means clustering.
The goal is to analyze customer purchasing behavior and generate useful business
insights for targeted marketing and decision-making.

>> Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

>> Steps Performed
1. Data preprocessing and cleaning
2. Exploratory Data Analysis (EDA)
3. Applying K-Means clustering
4. Visualizing customer segments

>>Result
Customers are grouped into different clusters based on income and spending score,
which helps businesses understand customer behavior and improve marketing strategy.
